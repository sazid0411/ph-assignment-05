Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an elements in the DOM. Here the some of the popular and best way of select an elements in the DOM:

  1. Select by TagName : getElementByTagName() , this method is used to select a DOM element by tag name of an element.
  2. Select by ID : getElementById() , this method is used to select a DOM element by ID of an element.
  3. Select by Class name : getElementsByClassName() , this method is used to select a DOM element by class name of an element.
  4. Select by CSS selector: querySelector() , this method is used to select a DOM element by CSS selector of an element.
  5. Multiple elements Select by CSS selector: querySelectorAll() , this method is used to select multiple DOM element by CSS selector of an element.
