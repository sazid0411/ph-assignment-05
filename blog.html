Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an elements in the DOM. Here the some of the popular and best way of select an elements in the DOM:

  • Select by TagName : getElementByTagName() , this method is used to select a DOM element by tag name of an element.
  • Select by ID : getElementById() , this method is used to select a DOM element by ID of an element.
  • Select by Class name : getElementsByClassName() , this method is used to select a DOM element by class name of an element.
  • Select by CSS selector: querySelector() , this method is used to select a DOM element by CSS selector of an element.
  • Multiple elements Select by CSS selector: querySelectorAll() , this method is used to select multiple DOM element by CSS selector of an element.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Difference between innerHTML, innerText, and textContent:

  1. innerHTML: Returns or sets the HTML contained within an element, including tags.
  2. innerText: Gets or sets the text content of an element, excluding hidden elements, taking CSS styles into account.
  3. textContent: Gets or sets all text in an element, including hidden elements, without any format.

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM is a technique where a parent element handles events of its child elements on the basis of event bubbling. Instead of adding event listeners to each child, a single listener is added to the parent, which is a performance improvement and code maintainability.

Question-4: What is event bubbling in the DOM?

Event Bubbling in the DOM occurs when an event starts at the target element and propagates upwards to the parent elements until the root (document) is reached. It moves from the innermost to the outermost. It makes parent elements listen to events triggered by their child elements.
